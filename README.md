## Project Overview
This project analyzes UK train rides data using **Power BI** to understand passenger demand, forecast future rides and revenue, and support business decision-making.

The project covers the full data analysis lifecycle:
- Data cleaning and preprocessing
- Data modeling
- Exploratory analysis
- Forecasting
- Dashboard design
- Business insights and recommendations

---

## Project Objectives
- Analyze train rides trends over time
- Forecast the number of rides for the next month
- Forecast expected revenue
- Analyze demand across different ticket classes
- Support decision-makers with clear insights
